Mengenal HTML, CSS, Javascript dan Tailwind

HTML adalah bahasa dasar untuk membuat struktur sebuah halaman web, seperti menulis judul, paragraf, gambar, dan tautan.

CSS berfungsi untuk mengatur tampilan web agar lebih menarik â€” mulai dari warna, ukuran huruf, tata letak, hingga efek animasi sederhana.

JavaScript digunakan untuk membuat web menjadi interaktif, misalnya menampilkan pesan, membuat tombol berfungsi, atau mengganti tema dengan sekali klik.

Tailwind CSS adalah alat bantu modern untuk mempercepat desain web. Kita bisa menata tampilan hanya dengan menambahkan class tanpa harus menulis kode CSS panjang.

Tata Cara Membuat Website Sederhana

  1. Persiapkan alat kerja. Gunakan teks editor seperti Visual Studio Code, Sublime Text, atau Notepad++. Pastikan juga sudah punya browser seperti Google Chrome atau Edge untuk menampilkan hasilnya.

  2. Buat folder proyek. Buat satu folder di komputer yang akan menjadi tempat semua file web disimpan. Misalnya, beri nama website-ku. Folder ini nantinya akan berisi file HTML, CSS, dan JavaScript.

  3. Buat file utama. Di dalam folder tersebut, buat file baru dengan nama index.html. File ini akan menjadi halaman utama website. Tulis struktur dasar HTML seperti <html>, <head>, dan <body>.

  4. Tambahkan konten. Di dalam <body>, tulis elemen-elemen seperti judul, paragraf, gambar, dan tautan agar websitemu punya isi. Contohnya: 
    <h1>Selamat Datang di Web Saya</h1>
<p>Ini adalah website pertama saya.</p>

  5. Tambahkan tampilan (CSS). Buat file baru bernama style.css dan hubungkan ke index.html menggunakan tag <link>. Gunakan CSS untuk mengatur warna background, ukuran teks, jarak antar elemen, dan gaya huruf agar tampilannya lebih menarik.

  6. Tambahkan interaksi (JavaScript). Buat file script.js lalu hubungkan ke HTML dengan tag <script>. JavaScript digunakan agar halamannya bisa melakukan aksi â€” misalnya, menampilkan pesan saat tombol diklik.

  7. Lihat hasilnya. Simpan semua perubahan, lalu buka file index.html di browser. Jika tampilannya sudah sesuai, berarti website telah berhasil dibuat! kita bisa terus mengedit dan menambahkan fitur sesuai ide dan kreativitas.
